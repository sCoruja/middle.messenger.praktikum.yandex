## Установка

```bash
# установка зависимостей
$ npm install / yarn

# запуск сервера
$ npm run start / yarn start

# запуск сервера в режиме разработки
$ npm run dev / yarn dev

# запуск линтера
$ npm run lint / yarn lint

# запуск тестов
$ npm run test / yarn test
```
